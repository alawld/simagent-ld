# Subterrans

A modern ant colony simulation game — a spiritual successor to SimAnt (1991) with a retro aesthetic and contemporary game design sensibilities.

## Tech Stack

To be finalized during the PRD phase. Current direction: TypeScript with Phaser 3 or PixiJS, with a clean separation between simulation logic and rendering.

## Directory Layout

- **src/** — Game source code. Simulation logic and rendering are kept separate so the core systems can be tested independently.
- **assets/** — Game assets organized by type:
  - `sprites/` — Sprite sheets, tilesets, and other visual assets
  - `audio/` — Sound effects and music
  - `fonts/` — Custom fonts
- **docs/** — Documentation for contributors: architecture guides, API references, and design explanations.
